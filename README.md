# 项目简介

## 核心功能
- 基于 YOLOv8 的牛行为检测系统，可识别牛的 5 种行为：饮水(drinking)、休息(resting)、休息段(resting bout)、站立(standing)、行走(walking)
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供用户界面，可实时显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供多种显示选项：可选择显示标签和原图
- 支持批量检测功能，可查看所有检测结果
- 包含 4600 张标注数据集及训练好的模型

## 适用场景
- 牧场自动化管理系统
- 牛只行为监测研究
- 畜牧业行为分析
- 农业智能化管理
- 动物行为学研究

## 技术特点
- 使用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 提供完整源码、标注数据集、训练好的模型及运行教程
- 支持多种开发环境：Pycharm+Anaconda 或 VSCode+Anaconda
- 包含 UI 界面源文件及图标资源
- 项目已配置成功环境，可直接使用
